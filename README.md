# 🐍 Miniguia de Estudos: Lógica de Programação com Python no NotebookLM

Este repositório foi desenvolvido como parte de um desafio prático da **DIO (Digital Innovation One)**. O projeto foca no uso da Inteligência Artificial (NotebookLM) para potencializar a aprendizagem ativa em lógica de programação, especificamente voltada para a linguagem Python.

---

## 🎯 Contexto e Objetivos

O foco principal deste caderno é o aprimoramento da **Lógica de Programação voltada ao Python**. 

**Meus objetivos de estudo são:**
*   Entender com maior profundidade como um computador "pensa" e executa instruções.
*   Aprimorar habilidades de **automação** de tarefas.
*   Construir uma base sólida para o aprendizado de futuras linguagens de programação, utilizando o Python como porta de entrada devido à sua simplicidade e poder.

## 📖 Miniguia de Estudo 

### 📝 Resumos Estruturados do Assunto

#### 1. O Pensamento Computacional
Programar é dividir um problema em três fases: **Entrada de dados** (captação), **Processamento** (análise e cálculos no cérebro da máquina) e **Saída** (exibição do resultado). Diferente de nós, computadores não têm intuição; eles precisam de passos detalhados e sem ambiguidades.

#### 2. Pilares da Lógica
*   **Variáveis e Tipos:** Como o computador armazena informações temporárias.
*   **Estruturas Condicionais (If/Else):** Tomadas de decisão baseadas em fatos ("Se saldo > valor, então saca").
*   **Estruturas de Repetição (Loops):** Automação de tarefas repetitivas (fazer algo "enquanto" uma condição for real).
*   **Modularização:** Não reinventar a roda, dividindo o código em funções menores.

### 📒 Glossário de Conceitos
*   **Algoritmo:** Uma sequência finita de passos para resolver um problema (como uma receita).
*   **Sintaxe:** As regras gramaticais rígidas de uma linguagem. Um erro aqui impede o programa de rodar.
*   **Semântica:** O significado real e a coerência do que foi escrito no código.
*   **Indentação:** Recuo do texto para organizar blocos de código (fundamental no Python).
*   **Debug:** Processo de encontrar e corrigir erros na lógica ou no código.
